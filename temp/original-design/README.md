Original Design Backup Reference
===============================

This folder preserves the original sample state before India-focused content updates.

How to restore original version:
1. Create a branch from this commit:
   git checkout -b original-sample 384bd84
2. Run the app in that branch:
   npm.cmd install
   npm.cmd run dev
